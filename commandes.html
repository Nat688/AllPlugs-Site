Commandes

Liste complète:

  • /reglement affiche le lien pour accedé au règlement.
  • /wiki ouvre l'interfaces du wiki.
  • /discord donne le lien du serveur discord.
  • /help ouvre l'interfaces du wiki.
  • /shop ouvre le shop, là où vous pouvez trouvé des spawners.
    • /sell hand vend l'object dans votre main.
    • /sellall [item] vend tous les objects séléctionné dans votre inventaire.
    • /sellall inventory vend tous les object dans votre inventaire.
  • /claim claim la zone autour de vous avec des claimblock.
    • /buyclaim 200$/claimbloc.
    • /sellclaim 20$/claimbloc.
    • /deleteallclaims supprime tous vos claims.
    • /deleteclaim supprime le claim où vous vous situez.
    • /claimlist vous donne tous vos claim (avec coordonné).
  • /msg [pseudo] [message] permet de msg un joueur.
  • /r [message] comme /msg mais avec la dernière personne à laquelle vous avez parlé.
  • /sb affiche ou non le scoreboard (truc à droite de votre écran)
  • /trash ouvre une poubelle.
  • /glow ouvre l'interface qui permet de vous appliquer glowing avec une couleur personalisé.
  • /chatcolor gui ouvre l'interface qui permet d'appliquer une couleur quand vous parlez dans le chat.

Premium ou voir /perms en jeu:

  • /craft ouvre la table de craft.
  • /heal vous redonne toute votre vie.
  • /feed vous redonne toute votre barre de faim.
  • /ec ouvre l'enderchest.
  • /fly vous permet de voler à l'infini.
  • /condense condense les items (ex: foin).
  • /spawn téléporte au spawn du serveur.
